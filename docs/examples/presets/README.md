# Presets Addon Example

This example demonstrates the optional `a11y-async-button/addons/presets` entry.

It shows:

- built-in `save`, `send`, and `delete` presets
- a project-defined `archive` preset
- shared `onAction` handling across preset-marked buttons
- visible demo status text listening to core state-change events

Run `npm run build` before opening the page so `docs/assets/addons/presets.js`
exists.
